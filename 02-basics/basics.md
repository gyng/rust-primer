# What is Rust?

- Systems language
  - C/C++ compatiability

- Functional features
  - pattern matching
  - `Option`/`Some`
  - traits/typeclasses
  - ADTs

- Safe by default
  - No `null`
  - stdlib APIs (eg, string APIs)
